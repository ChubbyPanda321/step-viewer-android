occt-import-js WASM Setup
==========================

The occt-import-js library provides STEP and IGES parsing via OpenCASCADE compiled to WebAssembly.

To install:

Option 1 - npm (recommended):
  1. npm install occt-import-js
  2. Copy from node_modules/occt-import-js/dist/:
     occt-import-js.js  →  app/src/main/assets/viewer/occt-import-js.js
     occt-import-js.wasm → app/src/main/assets/viewer/occt-import-js.wasm

Option 2 - GitHub releases:
  Download from: https://github.com/kovacsv/occt-import-js/releases
  Copy occt-import-js.js and occt-import-js.wasm to:
    app/src/main/assets/viewer/

Currently the occt-import-js.js is a PLACEHOLDER that shows an error
message when files are loaded. Replace it with the real file as above.
